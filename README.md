### Usage

Download & run

    curl -o wg-installer.sh https://raw.githubusercontent.com/jjinnee/wireguard-server-installer/main/install.sh && chmod +x wg-installer.sh && bash wg-installer.sh
    
You can start, stop, view, and remove with command.

    $ wghelp
